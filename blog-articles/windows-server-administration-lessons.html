Windows Server Administration: Lessons from a Real-World Internship

November 22, 2025
Mungu Jakisa Maurice
9 min read

Introduction

During my internship at NCR / SCI, I moved from theory to real enterprise systems. This article highlights what I learned in real operational Windows Server environments.

What I Worked On

  • Windows Server installation and configuration
  • Active Directory domain setup
  • User and group management
  • VMware virtual machines
  • Basic network and printer configuration

Understanding Active Directory

Centralized identity management, role-based access control, and proper structure and naming conventions. Shifted my perspective on security and scalability.

Virtualization with VMware

Understanding resource sharing, isolation, and snapshots — preparing me for cloud computing concepts.

Real Lessons You Don't Get in Class

  • Documentation is critical
  • One misconfiguration can affect many users
  • Always test before deploying

How This Shaped Me

Strengthened my interest in systems engineering, infrastructure, and reliability-focused development.
